# Animated Profile Dropdown UI
A clean, animated profile dropdown UI component built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just sharp CSS animations and crisp SVG icons.

## Features

- **Spinning gradient avatar ring** — conic-gradient border with continuous rotation animation
- **Spring dropdown entrance** — opens with a cubic-bezier bounce for a polished, native-app feel
- **Staggered menu items** — each item slides in with a sequenced delay on open
- **Icon micro-interactions** — icons scale up subtly on hover
- **PRO badge with pulse glow** — animated gradient badge on the Subscription item
- **Ripple click effect** — ink ripple on every menu item interaction
- **Danger styling** — Sign out item has distinct red color with pink hover state
- **Outside-click to close** — dropdown dismisses cleanly on any outside interaction
